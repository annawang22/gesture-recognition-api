This document records the AI tools and prompts used to build this project.

--------------------------------------------------------------------------------
CHATGPT PRO
--------------------------------------------------------------------------------

  1. Initial script
     "Could you write me a simple Python script that takes a picture on my
     webcam, sends that to U.S. Face++, and classifies the gesture or gives
     me some type of info on the gesture?"

  2. Externalize credentials
     "Instead of placing the API key and secret in the Python file, could you
     change it so it reads from a text file?"

  3. Add documentation
     "Could you also insert comments into the Python script explaining it so
     I understand what is going on?"

  4. Volume control via gesture
     "Can you build upon this code so that when I do a thumbs up gesture and
     it is detected, it simulates the volume up key on my keyboard
     (increasing the volume)?"

       API error: CONCURRENCY_LIMIT_EXCEEDED

  3. Build a GUI app
     "YES THANK GOD IT WORKS. Now can you help me build a cute Python app

     "It's still not working — I know that the camera is on but the app

  7. Simplify the app
     "Okay, can you remake me a simple Python app based on my code here?"
